How to Apply CSS to HTML

To include an external CSS file in an HTML file, place a link element in the head section: <link rel="stylesheet" href="styles.css">

How to Use JavaScript in a Webpage

To link to an external JavaScript file, place a script element at the bottom of the body section: <script src="scripts.js"></script> To include JavaScript directly in an HTML file, you can place the code within script tags in the body section: 

<script>\nalert('Hello, world!');\n</script>
